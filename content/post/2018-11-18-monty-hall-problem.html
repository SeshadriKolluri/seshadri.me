Problem Description

Monty Hall Problem is one of the most popular probability puzzles. The problem describes a game show where there are three doors. One of the doors contains a prize behind the door, and the other two doors don’t have the prize behind them.

The rules of the game are as the following.

  • You first choose a door and let the host know. (Original Choice, ◽).
  • The host knows which door has the prize, and opens one of the the doors that don’t have the prize behind them. This door is then eliminated. (Eliminated Choice, ◾).
  • You now have the option of sticking with your Original Choice, or changing to the other remaining door after elimination (Alternate Choice, ▦).

The question is whether you are better off sticking to your Original Choice, or switching to the Alternate Choice after the elimination, or if it doesn’t matter.

Solution

Apparently an overwhelming majority of the people think that it doesn’t matter whether you stick to your original choice or change your choice to the remaining door, as both of them are expected to have 33% chance of success originally, but it is now well known that the chances of success are better if we switch to the remaining door after elimination. We will try to understand it in a few different ways.

Approach 1 - A Thought Experiment

Let us assume we choose Door 3 initially. Now imagine Door 1 + Door 2 as one unit, and Door 3 as the second unit as shown below. We can see that the probability of success for the Door 1 + Door 2 unit is 2/3. Now, imagine if the host gives us the ability to choose that entire unit (consisting of two doors) instead of Door 3. We would obviously choose that unit as it has a 2/3 probability of success. When the host eliminates Door 2 and gives you an option to choose Door 1 (or he eliminates Door 1 and gives us the option to choose Door 2), he is effectively giving us an option to choose the entire unit of Door 1 + Door 2. So, when we leave our original choice (Door 3) behind, and switch to this other non-eliminated door in the two door unit, we have a higher probability of success (2/3).

While the above explanation may be intuitive enough for some people, it may still appear a little hand-waving. So, we will explore a couple more approaches below.

Approach 2 - Simulations

We can verify if logic from the above argument is true, by simulating this experiment a large number of times. In this section, we simulate the experiment 30 times (for simplicity), and use the visuals below to understand the results. The simulations and visuals were generated using R programming language. You can see the code by clicking on the code button below.

set.seed(2017)

# number of runs
n <- 30

# make a dataframe containing three columns called door1, door2, door3.
# Each row should contain two "goat"s and one "car", randomly distributed
# among three columns

car_positions <- sample(1:3, n, replace = TRUE)

# run the experiment chosen number of times
df_experiment_runs <-
  ldply(car_positions, function(x)
    ifelse(1:3 == x, "car", "goat"))
names(df_experiment_runs) <- c("Door1", "Door2", "Door3")
df_experiment_runs$`Run Number` <-
  as.numeric(row.names(df_experiment_runs))
df_experiment_runs$`First Choice` <- sample(1:3, n, replace = TRUE)
df_experiment_runs$`Prize Location` <- car_positions
df_experiment_runs$`Eliminated Position` <- apply(
  df_experiment_runs,
  MARGIN = 1,
  FUN = function(x) {
    (
      setdiff(1:3, c(x['Prize Location'], x['First Choice']))
        [sample(length(setdiff(1:3, c(x['Prize Location'], x['First Choice']))), 1)]
    )
  }
)

df_experiment_runs$`Alternate Choice` <- apply(
  df_experiment_runs,
  MARGIN = 1,
  FUN = function(x) {
    (
      setdiff(1:3, c(x['Eliminated Position'], x['First Choice']))
        [sample(length(setdiff(1:3, c(x['Eliminated Position'], x['First Choice']))), 1)]  
    )
    
  }
  
)

df_experiment_runs_long <- pivot_longer(df_experiment_runs,
                                        cols = starts_with("Door"),
                                        names_to = "Door")
df_experiment_runs_long$Door <-
  gsub('Door', '', df_experiment_runs_long$Door)

df_legend <-
  expand_grid(
    Choices = c(
      "Prize Location",
      "Original Choice",
      "Winning Original Choice",
      "Eliminated Choice",
      "Alternate Choice"
    )
  )
df_legend$Choices <- fct_inorder(df_legend$Choices)

# Generate a plot to be used for the legend.
get_legend_plot <- function(df) {
  ggplot(df, aes(x = Choices, fill = Choices, y = 1)) +
    geom_tile(color = 'gray50',
              width = 0.15,
              height = 1) +
    coord_cartesian(ylim = c(0, 2)) +
    facet_grid(~ Choices, scales = 'free') +
    geom_point(data = df %>% filter(grepl("Prize Location|Winning", Choices)), size = 0.5) +
    scale_fill_manual(values = c('white', 'gray80', 'gray80', 'gray30', 'bisque')) +
    theme_void() +
    
    theme(
      legend.position = 'none',
      axis.text.x = element_text(size = 9),
      strip.text = element_blank()
    )
}

Approach 3 - Exact Probability Calculations

Though the previous simulations make sense, they do not provide a definitive proof that switching our door choice would increase our probability of success to 2/3. As a matter of fact, there could be a different set of 30 simulation runs, where the results could be the opposite of what we got (though less likely).
